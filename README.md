# Clicons

A clean, open-source icon library built from the MIT-licensed HugeIcons set.

Currently includes **4559 icons** with full React support and a Lucide-like DX.

- **Available:** `clicons-react` – full React component library
- **Base package:** raw SVGs from `svg-icons`
- **Coming soon:** Svelte and Vue packages

Designed for **performance**, with **tree-shakeable**, **lightweight**, and **direct SVG rendering**—no runtime overhead.

For documentation and usage, see [clicons-react](./react/README.md).
