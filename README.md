# Clicons

A clean, open-source icon library built from the MIT-licensed HugeIcons set.

Currently includes **4434 icons** with full React and Svelte support and a Lucide-like DX.

- **Available:** 
  - `clicons-react` – full React component library  
  - `clicons-svelte` – full Svelte component library
- **Base package:** raw SVGs from `svg-icons`
- **Coming soon:** Vue package

Designed for **performance**, with **tree-shakeable**, **lightweight**, and **direct SVG rendering**—no runtime overhead.

For documentation and usage, see [clicons-react](./react/README.md) and [clicons-svelte](./svelte/README.md).
